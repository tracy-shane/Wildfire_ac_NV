# Wildfire_ac_NV
Wildfire Acres Burned in Nevada and Drought

I have been showing students the relationship between drought severity and normal/abovenormal years on wildfire acres burned in Nevada.

This is a project that allows me to update the figure with new data as it comes in each year.

The wildfire polygon data was downloaded from Nevada Department of Conservation and Natural Resournces Data Hub:  
Wildland Fire History (1910-2025): https://arcgis.water.nv.gov/enterprise/apps/sites/#/dcnr-open-data-hub/datasets/ae946d80525a4d02bbe42f139fe16f86/about

When fires burned across state lines from Nevada, those full fire acres were included in the acreage total.

The Drought Severity Coverage Index was downloaded from: https://droughtmonitor.unl.edu/DmData/TimeSeries.aspx
